MarkDown基础语法使用参考

标题

  • 语法
    一级标题用一个#，二级标题## ….最多展示六级标题

  • 效果

    一级标题

    二级标题

    三级标题

    四级标题

    五级标题
    六级标题

字体

  • 加粗

    • 语法
      要加粗的文本前后用两个 ** 符号包裹起来
    • 效果
      我是加粗后的字体

  • 斜体

    • 语法
      要倾斜的文本前后用一个 * 符号包裹起来
    • 效果
    • 我是倾斜后的字体*

  • 斜体&加粗
    *语法
    要倾斜并且加粗的文本前后用三个 *** 符号包裹起来
    *效果

  • 我是倾斜并且加粗后的字体*

  • 删除线

    • 语法
      要加删除线的文本左右分别用两个 ~~ 符号包起来
    • 效果
      我是添加删除线的字体

段落

  • 空格

    • 语法 或者使用全角空格。即：在全角输入状态下直接使用空格键就ok了
    • 效果
        插入一个空格文本
        插入两个空格文本
        插入四个空格文本
      　　全角按了两个空格文本😎

  • 换行

    • 语法
      使用 HTML 的 <br> 标签来实现换行。
    • 效果
      我在这里需要换行啦
      精彩内容在仍在继续…

块引用

  • 单个段落块引用

    • 语法
      要创建单个段落块引用，请在段落前添加一个 > 符号。
    • 效果

      我是单个段落引用

  • 多个段落块引用

    • 语法
      要创建单个段落块引用，请在段落前与段落之间添加 > 符号。
    • 效果

      我是段落一

      我是段落二

  • 嵌套块引用

  • 语法
    在要嵌套的段落前添加一个 >> 符号。

  • 效果

    我是段落一

    我是段落二并且需要嵌套引用

    无限嵌套引用…

列表

  • 有序列表

    • 语法
      列表项前添加数字并紧跟一个英文句点 1. 2. 3. 等等 …

    • 效果

    1.有序列表
    2.有序列表
    3.有序列表
    …

  • 无序列表
    为了兼容，请不要在同一个列表中混用不同的分隔符，最好选定一种分隔符使用

    • 语法
      无序列表用 - + * 任意一种都可以
    • 效果
      • 无序列表
      • 无序列表
      • 无序列表

代码

  • 代码块高亮

    • 语法
    • 效果

  • 单行代码块

    • 语法
      单行代码块使用 ` `包裹
    • 效果
      System.out.println("Hello World!");

图片

  • 语法
    [图片上传中…(image-9df155-1642153968831)]
    图片alt就是显示在图片下面的文字，相当于对图片内容的解释。
  • 效果

分割线

  • 语法
    为了兼容性，请在分隔线的前后均添加空白行
    三个以上 — 或 ***
  • 效果

超链接

  • 语法
    要创建链接，请将链接文本括在方括号（例如 [Duck Duck Go]）中，后面紧跟着括在圆括号中的 URL（例如 (https://duckduckgo.com) ）。
  • 效果
    My favorite search engine is Duck Duck Go.

  • 效果

      网址、电子邮件地址

  • 语法
    将 URL 或电子邮件地址括在<>里
  • 效果

转义字符

以下列出的字符都可以通过使用反斜杠字符从而达到转义目的。
字符|名称
–|–
| \ | 反斜杠
| * | 星号（asterisk） |
| _ | 下划线（underscore） |
| { } | 花括号（curly braces） |
| [ ] | 方括号（brackets） |
| < > | angle brackets |
| ( ) | 圆括号或括号（parentheses） |
| # | 井号（pound sign） |
| + | 加号（plus sign） |
| - | 减号（minus sign） (也叫连字符 hyphen) |
| . | 句点（dot） |
| ! | 感叹号（exclamation mark） |
